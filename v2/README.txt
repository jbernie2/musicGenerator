To Run This Project
move to directory musicGenerator and

To run the project with Maven:
1. Build Project:
	mvn package
2. Run Code:
	mvn exec:java

To run the project without Maven:
1. Navigate to 
	v2/musicGenerator/src/main/java/com/john-bernier/musicGenerator
2. Compile code:
	javac App.java
3. Run Code:
	java App

To hear the results listen to midifile.mid in v2/musicGenerator directory
